# Tags (extended) plugin

A plugin for [Kirby CMS](http://getkirby.com) which adds support for tags listing to the original plugin.

## Kirby installation
Put the `tag/` folder in `/site/plugins/`.

## Options
```YAML
fields:
  tags:
    label: Tags
    type: tags
    separator: /
    list: show | toggle | hide
    limit: 2
```
